\# BioSys\_Project



This repository contains the final vignette developed for the BioSys course.



\## Description



The vignette demonstrates the use of R for spatial transcriptomics analysis and visualization, focusing in cell-cell communication.



\## Repository structure



\- `CellChat\_ProximityNetwork\_PathwaySpace.Rmd`: main R Markdown file.

\- `bibliography.bib`: references cited in the vignette.



\## Requirements



The analysis was written in R and requires the following packages:



\- ggplot2

\- igraph

\- CellChat

\- RGraphSpace

\- Seurat
